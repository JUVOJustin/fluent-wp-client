## Overview

Root package exports are Standard Schema validators — validator-agnostic. Mutations auto-validate with built-in schemas. Pass your own schema to override.

:::note
Schemas are authored in Zod internally, but root exports expose only the Standard Schema contract.
:::

## Which import path?

| Need | Path |
|---|---|
| Extend schemas (`.parse()`, `.extend()`, `.shape`) | `fluent-wp-client/zod` |
| Validator-agnostic code | `fluent-wp-client` (root) |
| Custom validator (Valibot, ArkType) | Pass your own schema to `responseSchema` / `inputSchema` |
| CPT resource validation | `content('books', schema)` or `terms('genre', schema)` |

:::tip
`content(resource, schema)` and `terms(resource, schema)` validate all reads and mutations on that resource client.
:::

## Extend with Zod

```ts
import { postSchema } from 'fluent-wp-client/zod';

const extendedSchema = postSchema.extend({
  acf: z.object({
    acf_subtitle: z.string().optional(),
  }).optional(),
});

const created = await wp.content('posts').create(
  { title: 'Validated', status: 'draft' },
  extendedSchema,
);
```

## Valibot

```ts
import * as v from 'valibot';

const created = await wp.content('posts').create(
  { title: 'Valibot', status: 'draft' },
  v.object({ id: v.number(), slug: v.string(), status: v.string() }),
);

// CPT with Valibot
const books = wp.content('books', v.object({
  id: v.number(), slug: v.string(), status: v.string(),
  title: v.object({ rendered: v.string() }),
}));
```

## Sparse CPT validation

```ts
import { postLikeWordPressSchema, contentWordPressSchema } from 'fluent-wp-client';

const artifacts = wp.content('artifacts', postLikeWordPressSchema); // tolerates missing fields
const strict = wp.content('artifacts', contentWordPressSchema);     // fails if fields missing
```

## Astro Actions

:::note
Astro actions require native Zod input schemas.
:::

```ts
import { defineAction } from 'astro:actions';
import { postWriteBaseSchema } from 'fluent-wp-client/zod';

export const server = {
  createPost: defineAction({ input: postWriteBaseSchema, handler: async (input) => input }),
};
```

## Manual Standard Schema validation

```ts
import { postSchema } from 'fluent-wp-client';

const result = await postSchema['~standard'].validate(payload);
if (result.issues) throw new Error('Validation failed');
const post = result.value;
```
